# CMPE 257 Final Project

This project aims to predict the financial success of films by classifying them as either a **Hit** or a **Flop** based on publicly available metadata. By leveraging the **TMDB 5000 Movies Dataset**, the study explores which features most significantly contribute to a movie's commercial performance.

## Project Goal
The primary objective is to build and compare multiple machine learning models to determine their effectiveness in predicting cinematic success. Beyond model accuracy, the project focuses on best practices in machine learning, including:
* Robust data preprocessing and handling diverse data types.
* Preventing data leakage to ensure realistic performance estimates.
* Evaluating feature importance to understand the drivers of movie profitability.

## Machine Learning Models
We implement and evaluate the following classifiers using the Scikit-Learn library:
* **Logistic Regression**
* **Random Forest**
* **Support Vector Machine (SVM)**
* **Gradient Boosting (XGBoost)**

## Evaluation Metrics
To provide a comprehensive analysis of model performance, we utilize:
* Accuracy, Precision, Recall, and F1-Score
* ROC-AUC
* Confusion Matrices
* Cross-Validation

## Dataset
The project uses the [TMDB 5000 Movies Dataset](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata), which includes metadata such as budget, genres, runtime, and cast/crew information.
